## Description
Given a collection of intervals, merge all overlapping intervals.
## Example
### Example 1
<pre>
Input: [(1,3)]
Output: [(1,3)]
</pre>
### Example 2
<pre>
Input:  [(1,3),(2,6),(8,10),(15,18)]
Output: [(1,6),(8,10),(15,18)]
</pre>

### Challenge
O(n log n) time and O(1) extra space.
